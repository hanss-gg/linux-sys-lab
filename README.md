# Linux Sysadmin Lab

Local Linux sysadmin lab simulating real-world server administration tasks.
Built as part of a cloud engineering learning path.

## Topics Covered
- User & permission management
- SSH configuration
- Web server setup (Nginx)
- Firewall rules (UFW)
- Bash scripting

## Structure
- `scripts/` — automation bash scripts
- `docs/` — notes and documentation per topic
